Set the following in .env.local based on your twitter dev acct:

```
TWIT_CONSUMER_KEY=...
TWIT_CONSUMER_SECRET=...
TWIT_ACCESS_TOKEN=...
TWIT_ACCESS_SECRET=...
```

and then run `npm run dev`
